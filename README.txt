Name: Biotop Analyzer

Authors: Ing. Josef Pavlíček, Ph.D. and Ing. Adam Tetaur

Launch and use: Launch by pressing BiotopAnalyzer.exe and load image you want to analyze. You can choose four separate coulour filters, or their combination, to visualize analysis subject in different coulour scales with AplikujFiltr (apply filter) button. 
                Pressing "Show and save histogram" button will create output.csv file in the 'debug' folder. In there , you will get desired number of pixels representing coulour codes on the 0 - 255 scale. 0 representsv black and 255 represents white.

If you encounter any bugs, please, inform us so we can improve the software! 

Thank you and enjoy!
